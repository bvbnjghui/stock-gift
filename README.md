# A4 圖片排版工具

一個純前端的網頁工具，可將多張圖片自動排版成 A4 横式版面（4 列 × 2 行），並支援加入股票代號相關的中繼資料。

## 功能特色

- **圖片上傳**：一次選擇 1-8 張圖片，自動排列於 A4 横式版面
- **即時預覽**：上傳後自動生成預覽，支援滑鼠滾輪縮放與拖曳平移
- **中繼資料疊加**：上傳 CSV 檔案，根據股票代號自動在圖片上方顯示時間與地點
- **下載**：將排版結果輸出為高品質 JPG 圖片（3508×2480 像素，約 300 DPI）
- **列印**：直接透過瀏覽器列印功能輸出 A4 排版

## 使用方式

### 1. 開啟工具

直接在瀏覽器中開啟 `index.html` 即可使用，無需安裝任何後端環境或額外套件。

### 2. 選擇圖片

1. 在「選擇圖片」區域點擊檔案選擇框
2. 選取 1-8 張圖片（jpg、png 等常見格式）
3. 選擇後，每張圖片會對應一個「股票代號輸入」欄位

### 3. 設定輸出名稱與間距

- **建議輸出檔名**：輸入下載檔案的名稱（預設為 `a4_collage_web.jpg`）
- **圖片間距 (像素)**：設定圖片之間的空白距離，預設為 20 像素

### 4. 上傳中繼資料（CSV）

若要在排版上顯示時間與地點資訊，請準備一個 CSV 檔案，格式如下：

```csv
ticker,time,location
2330,09:30,台北
2317,10:00,新竹
2454,10:30,台中
```

欄位說明：
- `ticker`：股票代號
- `time`：顯示時間
- `location`：顯示地點

上傳後，系統會根據您在步驟 2 中輸入的股票代號，自動對應並疊加資訊於圖片上方。

### 5. 輸入股票代號

在「股票代號輸入」區域中，為每張圖片輸入對應的股票代號（或任意識別文字），以便與 CSV 中繼資料配對。

### 6. 預覽與操作

- 版面會自動於下方預覽區域顯示
- **縮放**：將游標移到預覽圖片上，使用滑鼠滾輪可放大/縮小
- **平移**：按住滑鼠左鍵拖曳可移動檢視位置
- **手機/平板**：支援單指拖曳與雙指縮放

### 7. 下載或列印

- **下載排版**：將 A4 排版儲存為 JPG 圖片
- **列印排版**：開啟列印預覽視窗，可直接印出 A4 尺寸排版

## 技術規格

| 項目 | 規格 |
|------|------|
| 版面尺寸 | A4 横式（3508 × 2480 像素） |
| 解析度 | 300 DPI |
| 版面配置 | 4 列 × 2 行，最多 8 張圖片 |
| 輸出格式 | JPEG（品質 90%） |
| 中繼資料編碼 | 支援 UTF-8 與 Big5 編碼的 CSV |

## 專案結構

```
stock/
├── index.html    # 主頁面
├── script.js     # 核心邏輯：圖片載入、排版生成、預覽、下載、列印
└── style.css     # 響應式樣式，支援手機、平板與桌機
```

## 瀏覽器建議

請使用現代瀏覽器（Chrome、Edge、Firefox、Safari）的最新版本開啟，以獲得完整的互動功能。

## 注意事項

- 若 CSV 檔案使用 Big5 編碼（常見於 Windows 環境），系統會自動偵測並正確解析
- 若選擇超過 8 張圖片，系統會自動只使用前 8 張
- 列印時請在印表機設定中選擇 A4 横式，並關閉「適應頁面」或類似選項，以確保實際尺寸正確
